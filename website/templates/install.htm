  1. Install Node Process
    • Ubuntu/Debian: node-i386.deb - (you probably want this one)
    • Ubuntu/Debian: node-amd64.deb 
      sudo dpkg -i node-i386.deb
or
sudo dpkg -i node-amd64.deb
    • Or Download NodeJS and compile from source: 
      ./configure && sudo make install

    node
    Juice

      tar xvfj nodejuice.bz2

      cd nodejuice && ./nodejuice website/
