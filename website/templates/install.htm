This is a quick tutorial on how to install
node
Juice
. Watch the video Installation HOW TO.
  1. Download Node (v0.1.27)
    • Then compile from source: 
      tar xvfz node-v0.1.27.tar.gz
cd node-v0.1.27
./configure && sudo make install
  2. Download
    node
    Juice
    -1.4.1
  3. Extract files then Start WSGI, Seeker and Browser:
    • Extract files: 
      tar xvfz stephenlb-nodejuice-#######.tar.gz
    • Start the server: 
      cd stephenlb-nodejuice-####### && ./nodejuice demo/rad/
    • Run Browser: 
      http://localhost:8080/
    • To Stop NodeJuice press control key and the letter 'c': 
      CTRL + C
    • To run on your own files: 
      cd stephenlb-nodejuice-####### && ./nodejuice /path/to/my/files
